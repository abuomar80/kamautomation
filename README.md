# MEDAD Automation Toolkit

## Overview
- Streamlines MEDAD tenant provisioning and configuration.
- Bundles curated automation flows for onboarding, maintenance, and verification.
- Designed for internal operations teams; source is mirrored publicly for transparency only.

## Included Capabilities
- Basic tenant bootstrap (policies, locations, templates, permission bundles).
- Advanced maintenance utilities (analytics scaffolding, MARC assistance, SIP2/Z39.50 helpers).
- Tenant cloning, reporting exports, and user management shortcuts.

## Operational Use
- Intended for MEDAD / Naseej staff and trusted partners under existing agreements.
- Follow internal runbooks and knowledge base articles for execution steps.
- Store credentials securely and purge local session data after each use.

## Security & Compliance
- Respect customer data boundaries and observe MEDAD security policies.
- Validate uploads before processing; never commit secrets to this repository.
- Report suspected vulnerabilities to the KAM support group immediately.
